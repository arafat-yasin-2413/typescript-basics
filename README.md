## Project Setup

### 1. Create a folder. Open it. And run : 
```bash
pnpm init
``` 
or 
```bash
npm init --y
```
`it will add the package.json file`

---

### 2. Add typescript in dev depandency: 
```bash
pnpm add -D typescript tsx
```

---

### 3. TypeScript configuration: 
```bash
npx tsc --init
```

--- 
### 4. To convert to js:
```bash
npx tsc
```
